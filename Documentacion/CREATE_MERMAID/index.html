Generador de Diagramas ER con Mermaid

Introducción

El Generador de Diagramas ER con Mermaid es una aplicación web fácil de usar que permite a los usuarios crear, visualizar y generar código Mermaid para diagramas entidad-relación (ER). Los usuarios pueden agregar tablas, definir atributos y establecer relaciones con diferentes cardinalidades. La aplicación proporciona una interfaz moderna para una fácil gestión de tablas, atributos y relaciones, lo que la convierte en una herramienta conveniente para modelar y visualizar estructuras de bases de datos.

La aplicación es compatible con M-TO-SQL, otra herramienta desarrollada por el mismo autor, que permite convertir los diagramas ER generados en código SQL para la creación de bases de datos.

Funcionalidades

  • Crear y eliminar tablas
  • Agregar y eliminar atributos a las tablas
  • Establecer relaciones entre tablas con diferentes cardinalidades
  • Generar código Mermaid a partir de las tablas y relaciones definidas
  • Visualizar el código Mermaid generado en un diagrama ER

Uso de la aplicación

Para utilizar la aplicación, sigue estos pasos:

  1. En la sección "Tablas", ingresa el nombre de una tabla y haz clic en "Agregar tabla" para crearla.
  2. En la sección "Atributos", selecciona la tabla a la que deseas agregar atributos en el desplegable "Tabla" y luego ingresa el nombre y el tipo del atributo en los campos correspondientes. Haz clic en "Agregar atributo" para agregarlo a la tabla.
  3. En la sección "Relaciones", selecciona las tablas que deseas relacionar en los desplegables "Tabla 1" y "Tabla 2". Luego, elige las cardinalidades de las relaciones en los desplegables "Cardinalidad Tabla 1" y "Cardinalidad Tabla 2". Finalmente, ingresa el texto de la relación y haz clic en "Agregar relación" para establecer la relación entre las tablas.
  4. Para generar el código Mermaid, haz clic en "Generar Código Mermaid". El código se mostrará en la sección "Código Mermaid".

Compatibilidad con M-TO-SQL

El Generador de Diagramas ER con Mermaid es compatible con la aplicación M-TO-SQL, lo que permite aprovechar los diagramas ER creados para generar código SQL para la creación de bases de datos. Después de generar el código Mermaid en esta aplicación, puedes exportarlo y utilizarlo en M-TO-SQL para generar el código SQL correspondiente.
